# Happy flow steps

## place input file in WordFilterer.CLI/Data/Data/Input

## open a terminal in WordFilterer.CLI

## execute command: dotnet run

## menu will open and ask two input variables:
	- A choice wether to generate combinations of two words (Y), any number of words (N), quit (q)
	- A length of which words you want to find combinations for (positive number)

## output file will be generated in WordFilterer.CLI/Data/Data/Ouput
